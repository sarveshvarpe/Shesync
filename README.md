# SheSync

## Empowering Women Through Safety, Health, and Smart Support

SheSync is an AI-powered women-centric platform designed to provide safety assistance, health support, emergency help, smart tracking, and seamless community connectivity in one application. The platform focuses on creating a secure and supportive ecosystem for women by integrating modern technologies such as AI, automation, real-time location services, and intelligent alerts.

---

# Features

## 1. Emergency SOS System

* One-tap emergency alert system
* Sends live location to trusted contacts
* Real-time emergency notifications
* Quick access during unsafe situations

## 2. Live Location Tracking

* Real-time GPS location sharing
* Safe route assistance
* Continuous location updates for emergency contacts

## 3. AI-Powered Assistance

* Smart chatbot support
* Instant responses for help and guidance
* Personalized assistance using AI integration

## 4. Women Health Support

* Health reminders and tracking
* Wellness guidance
* Smart notifications for important health-related activities

## 5. Secure Authentication

* User login and registration system
* Protected user data
* Secure authentication flow

## 6. Community & Support

* Connect with trusted users
* Share important updates
* Build a safe and supportive network

# Installation Guide

## Clone the Repository

```bash
git clone https://github.com/sarveshvarpe/shesync.git
```

## Navigate to Project Folder

```bash
cd shesync
```

## Install Dependencies

### Frontend

```bash
cd Shesync-web
npm install
```

### Backend

```bash
cd Shesync
pip install -r requirement.txt
```

---

# Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
DATABASE_URL=your_database_connection_url
JWT_SECRET=your_jwt_secret
REDIS_URL=your_redis_url
BREVO_API_KEY=your_brevo_api_key
EMAIL_FROM=your_email_address
GROQ_API_KEY=your_groq_api_key
```

---

# Run the Project

## Start Backend

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## Start Frontend

```bash
npm run dev --host
```
